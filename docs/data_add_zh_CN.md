# 站点与线路 ID 获取及导入说明

## 背景说明
游戏的底层代码使用**编号 (ID)** 来识别站点和线路。因此，在导入时刻表前，需要先从游戏中获取这些 ID 并录入本系统。

## 获取 ID 操作步骤

1.  **开启游戏调试模式**
    *   进入游戏，打开 **设置 → 高级**。
    *   启用 **调试模式**。
    *   **重启游戏** 以使设置生效，然后加载你的存档。

2.  **打开控制台并执行代码**
    *   在游戏中按 **\`** 键（反引号键）打开控制台。（此按键可在游戏设置中自定义）。
    *   将提供的专用代码复制到控制台中，按 **回车** 执行。

3.  **复制并处理输出数据**
    *   执行后，控制台会输出大量数据。
    *   请**复制所有文本**，范围从 `uil` 字样上方开始，一直到你粘贴的代码行之前。
    *   **正确的数据格式** 应为：一行名称、一行空格、一行数字（ID）、一行空格…… 如此循环。
    *   **注意**：名称中出现乱码是正常现象，不影响 ID 的准确性。

4.  **导入数据到软件**
    *   将上一步复制的全部文本，粘贴到本软件界面的**左侧输入框**中。
    *   根据你的需求，点击 **“站点导入”** 或 **“线路导入”** 按钮。
    *   **关于线路**：软件支持对长线路进行截断操作，具体请参考“截断选项”的说明。

## 重要注意事项

*   **数据一致性至关重要**：后续进行“同步时刻表”操作时，时刻表数据中的**站点名**和**线路名**，必须与这里导入的数据**完全匹配**（仅允许前后有空格）。
*   **确保 ID 正确识别**：无论是直接修改时刻表文件，还是修改生成的 `_station.xlsx` 或 `_line.xlsx` 文件，最终目的都是为了让软件能通过名称**正确匹配到游戏内部的 ID**。如果遇到不匹配的情况，请自行核对并修改相关数据。
